## General info
A simplified version of the browser game Ogame. The project is made in hexagonal architecture.
	
## Technologies
Project is created with:
* Java v 11
* Lombok v 1.18.18
* Passay v 1.6.0
* Spock v 2.0-M4-groovy-3.0
* Spring Boot v 2.4.2 (Data, Validation, Security)
* vavr v 0.10.3
* PostgreSQL
